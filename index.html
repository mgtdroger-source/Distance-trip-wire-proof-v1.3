Distances to Lines v2.8

v2.8 keeps the proven GPS engine behaviour and simplifies manual route entry. Waypoint, line start, and line end coordinates are now entered as combined lat/lon strings, using either a space or comma between latitude and longitude. Full GPS Log CSV remains unchanged.
Default test route: Bluefin line → Noordhoek Sign → Ruby Way → 90 Sign.

1. GPS / Run Control

GPS starts when the app opens. Wait for a stable GPS status, press Start Run before the first waypoint, then allow the 30 second run countdown.

2. Route Status

Field-use confirmation while driving: shows the active waypoint, crossed/waiting status, route distance, line result, and waypoint/radius comparison.

3. Crossing Results

For waypoints with line data, line crossing advances to the next waypoint. For point-only waypoints, radius entry advances to the next waypoint.

4. Export

The detailed diagnostic log is kept inside the CSV export. The Full GPS Log CSV remains untouched.

5. Route Waypoints / Manual Entry

Pre-run setup area. Enter coordinates as latitude/longitude in one field, for example -34.118409 18.382709 or -34.118409, 18.382709. No mode selector is needed: line data present means line crossing is the trigger; line data blank means radius trigger.
